Project on Console

Visual Studio Code is an offline code editor that helps you perform tasks and operations. As it is offline, you could code while not having to use internet. Brackets is a similar application but Visual Studio Code is much advanced for complex codings.

To set up VSC, you have to download it first and then install the .exe file for VSC. Then after few minutes of launching, create a new folder for your code and start off with an 'index.html' file!

A Console Screen usually takes input from the code and displays the output on the command-line window. It is used to represent or show the user the input, output and errors. To open the console screen, you have to go live first and then right click, which leads you to inspect. In the command-line window, you need to go to the 'Console' tab where you can see the output. To print console, you can use 'console.log' after creating a function/variable.
